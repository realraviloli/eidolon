# Eidolon License Server

Сервер лицензирования с админ-панелью.

## Деплой на Railway

1. Форкните этот репозиторий
2. Подключите Railway к GitHub
3. Создайте новый проект из репозитория
4. Railway автоматически определит Python и запустит сервер

## Переменные окружения (опционально)

- `PORT` - порт (по умолчанию 8080, Railway устанавливает автоматически)
